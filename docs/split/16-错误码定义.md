## 31. 错误码定义

### 31.1 错误码规范

错误码采用 **5位数字编码**，格式为：`XXYYY`

- **XX**：模块编码（2位）
- **YYY**：错误序号（3位）

| 模块编码 | 模块名称 |
|----------|----------|
| 10 | 系统级错误 |
| 11 | 认证授权错误 |
| 12 | 参数校验错误 |
| 20 | 用户管理 |
| 21 | 组织管理 |
| 30 | 销售管理 |
| 31 | 采购管理 |
| 32 | 库存管理 |
| 33 | 财务管理 |
| 40 | 生产管理 |
| 41 | 工序管理 |
| 42 | 扎包管理 |
| 43 | 质量管理 |
| 44 | 工资管理 |
| 50 | 基础数据 |

### 31.2 系统级错误码（10000-19999）

| 错误码 | 错误信息 | 说明 |
|--------|----------|------|

### 31.12 统一响应格式

```json
{
  "code": 0,
```

```json
{
```
