### 35.2 材料成本归集算法

#### 35.2.1 BOM展开算法

```
┌──────────────────────────────────────────────────────────────────────────────┐
│                            BOM展开算法                                        │
├──────────────────────────────────────────────────────────────────────────────┤
│                                                                               │
│  输入：产品ID、生产数量                                                       │
│  输出：所需材料清单（含数量、损耗率）                                         │
│                                                                               │
│  算法步骤：                                                                    │
│  1. 获取产品BOM结构树                                                         │
│  2. 递归展开所有层级子件                                                      │
│  3. 计算各材料需求数量 = 生产数量 × 单位用量 × (1 + 损耗率)                   │
│  4. 汇总相同材料的总需求                                                      │
│                                                                               │
└──────────────────────────────────────────────────────────────────────────────┘
```

**BOM展开SQL示例：**

```sql
WITH RECURSIVE bom_tree AS (
    -- 基础查询：获取产品的直接BOM
    SELECT 
        b.style_id AS parent_id,
        b.component_id,
        b.component_name,
        b.quantity,
        b.loss_rate,
        b.level,
        1 AS depth
    FROM jxc_bom b
    WHERE b.style_id = :styleId
      AND b.is_deleted = 0
    
    UNION ALL
    
    -- 递归查询：展开子件BOM
    SELECT 
        bt.component_id AS parent_id,
        b.component_id,
        b.component_name,
        bt.quantity * b.quantity AS quantity,
        b.loss_rate,
        b.level,
        bt.depth + 1
    FROM bom_tree bt
    JOIN jxc_bom b ON bt.component_id = b.style_id
    WHERE b.is_deleted = 0
      AND bt.depth < 10
)
SELECT 
    component_id,
    component_name,
    SUM(quantity * (1 + loss_rate)) AS total_quantity,
    MAX(depth) AS max_depth
FROM bom_tree
GROUP BY component_id, component_name
ORDER BY max_depth, component_name;
```

**领料汇总SQL示例：**

```sql
SELECT 
    pp.id AS production_id,
    pp.order_no AS production_no,
    pp.style_id,
    s.style_code,
    s.style_name,
    ppd.sku_id,
    ppd.sku_code,
    ppd.sku_name,
    SUM(ppd.quantity) AS total_quantity,
    SUM(ppd.quantity * ppd.unit_cost) AS total_cost,
    AVG(ppd.unit_cost) AS avg_unit_cost
FROM jxc_production_pick pp
JOIN jxc_production_pick_detail ppd ON pp.id = ppd.pick_id
JOIN jxc_style s ON pp.style_id = s.id
WHERE pp.is_deleted = 0
  AND pp.pick_status = 'completed'
  AND pp.pick_date BETWEEN :startDate AND :endDate
  AND pp.account_period = :accountPeriod
GROUP BY pp.id, pp.order_no, pp.style_id, s.style_code, s.style_name, 
         ppd.sku_id, ppd.sku_code, ppd.sku_name;
```

#### 35.2.3 材料成本归集

**归集算法流程：**

```
1. 确定核算期间
   └── 获取当期所有已完工/进行中的生产订单

2. 汇总领料数据
   └── 按生产订单汇总材料成本

3. 材料成本分配
   ├── 直接材料：直接归集到对应产品
   └── 共用材料：按分摊系数分配

4. 生成材料成本归集单
   └── 记录归集结果，作为成本计算依据
```

**材料成本归集SQL示例：**

```sql
INSERT INTO jxc_cost_material_collect (
    collect_no, account_period, production_id, style_id,
    material_cost, collect_date, collect_status,
    create_time, create_by
)
SELECT 
    CONCAT('MC', DATE_FORMAT(NOW(), '%Y%m%d'), LPAD(IFNULL(MAX(id), 0) + 1, 4, '0')) AS collect_no,
    :accountPeriod AS account_period,
    pp.id AS production_id,
    pp.style_id,
    SUM(ppd.quantity * ppd.unit_cost) AS material_cost,
    NOW() AS collect_date,
    'pending' AS collect_status,
    NOW() AS create_time,
    :userId AS create_by
FROM jxc_production_pick pp
JOIN jxc_production_pick_detail ppd ON pp.id = ppd.pick_id
WHERE pp.is_deleted = 0
  AND pp.pick_status = 'completed'
  AND pp.account_period = :accountPeriod
GROUP BY pp.id, pp.style_id;
```

#### 35.3.2 工时分摊算法

```
┌──────────────────────────────────────────────────────────────────────────────┐
│                            工时分摊算法                                       │
├──────────────────────────────────────────────────────────────────────────────┤
│                                                                               │
│  适用场景：计时工资、间接人工成本分摊                                        │
│                                                                               │
│  分摊公式：                                                                   │
│  产品人工成本 = 总人工成本 × (产品工时 / 总工时)                             │
│                                                                               │
│  工时类型：                                                                   │
│  - 直接工时：直接用于产品生产                                                │
│  - 间接工时：辅助生产、管理工时                                              │
│  - 机器工时：机器运转时间（用于机器折旧分摊）                                │
│                                                                               │
└──────────────────────────────────────────────────────────────────────────────┘
```

**工时分摊SQL：**

```sql
WITH work_hour_total AS (
    SELECT 
        SUM(wh.work_hours) AS total_hours,
        SUM(wh.wage_amount) AS total_wage
    FROM jxc_work_hour wh
    WHERE wh.is_deleted = 0
      AND wh.work_date BETWEEN :startDate AND :endDate
      AND wh.account_period = :accountPeriod
)
SELECT 
    wh.production_id,
    wh.style_id,
    s.style_code,
    s.style_name,
    SUM(wh.work_hours) AS product_hours,
    (SELECT total_hours FROM work_hour_total) AS total_hours,
    SUM(wh.wage_amount) AS direct_wage,
    (SELECT total_wage FROM work_hour_total) * 
        (SUM(wh.work_hours) / (SELECT total_hours FROM work_hour_total)) AS allocated_wage
FROM jxc_work_hour wh
JOIN jxc_style s ON wh.style_id = s.id
WHERE wh.is_deleted = 0
  AND wh.work_date BETWEEN :startDate AND :endDate
  AND wh.account_period = :accountPeriod
GROUP BY wh.production_id, wh.style_id, s.style_code, s.style_name;
```

### 35.4 制造费用分摊算法

#### 35.4.1 分摊方法概述

| 分摊方法 | 适用场景 | 分摊依据 | 优缺点 |
|----------|----------|----------|--------|
| 产量法 | 产品结构相似 | 完工产量 | 简单，但可能不准确 |
| 工时法 | 劳动密集型 | 生产工时 | 合理，计算复杂 |
| 机器时法 | 设备密集型 | 机器工时 | 准确，需设备记录 |
| 直接材料法 | 材料成本主导 | 材料成本 | 简单，适用性有限 |

#### 35.4.2 产量法分摊

```
┌──────────────────────────────────────────────────────────────────────────────┐
│                            产量法分摊算法                                     │
├──────────────────────────────────────────────────────────────────────────────┤
│                                                                               │
│  分摊公式：                                                                   │
│  产品制造费用 = 总制造费用 × (产品产量 / 总产量)                             │
│                                                                               │
│  适用条件：                                                                   │
│  - 产品工艺相似                                                              │
│  - 生产周期相近                                                              │
│  - 单位产品消耗资源相近                                                      │
│                                                                               │
└──────────────────────────────────────────────────────────────────────────────┘
```

**产量法分摊SQL：**

```sql
WITH production_total AS (
    SELECT SUM(output_quantity) AS total_quantity
    FROM jxc_production_order
    WHERE is_deleted = 0
      AND account_period = :accountPeriod
      AND order_status IN ('completed', 'partial_completed')
)
SELECT 
    po.id AS production_id,
    po.style_id,
    s.style_code,
    s.style_name,
    po.output_quantity,
    (SELECT total_quantity FROM production_total) AS total_quantity,
    :totalManufacturingCost * (po.output_quantity / (SELECT total_quantity FROM production_total)) 
        AS allocated_cost
FROM jxc_production_order po
JOIN jxc_style s ON po.style_id = s.id
WHERE po.is_deleted = 0
  AND po.account_period = :accountPeriod
  AND po.order_status IN ('completed', 'partial_completed')
ORDER BY s.style_code;
```

#### 35.4.3 工时法分摊

```
┌──────────────────────────────────────────────────────────────────────────────┐
│                            工时法分摊算法                                     │
├──────────────────────────────────────────────────────────────────────────────┤
│                                                                               │
│  分摊公式：                                                                   │
│  产品制造费用 = 总制造费用 × (产品工时 / 总工时)                             │
│                                                                               │
│  工时来源：                                                                   │

│  - 机器运转工时                                                              │
│  - 辅助生产工时                                                              │
│                                                                               │
│  费用类型：                                                                   │
│  - 折旧费                                                                    │
│  - 水电费                                                                    │
│  - 车间管理人员工资                                                          │
│  - 低值易耗品                                                                │
│  - 修理费                                                                    │
│                                                                               │
└──────────────────────────────────────────────────────────────────────────────┘
```

**工时法分摊SQL：**

```sql
WITH work_hour_summary AS (
    SELECT 
        production_id,
        style_id,
        SUM(work_hours) AS total_hours
    FROM jxc_work_hour
    WHERE is_deleted = 0
      AND account_period = :accountPeriod
    GROUP BY production_id, style_id
),
total_hours AS (
    SELECT SUM(total_hours) AS grand_total
    FROM work_hour_summary
)
SELECT 
    whs.production_id,
    whs.style_id,
    s.style_code,
    s.style_name,
    whs.total_hours,
    (SELECT grand_total FROM total_hours) AS grand_total_hours,
    :totalManufacturingCost * (whs.total_hours / (SELECT grand_total FROM total_hours)) 
        AS allocated_cost
FROM work_hour_summary whs
JOIN jxc_style s ON whs.style_id = s.id
ORDER BY s.style_code;
```

#### 35.4.4 机器时法分摊

```
┌──────────────────────────────────────────────────────────────────────────────┐
│                            机器时法分摊算法                                   │
├──────────────────────────────────────────────────────────────────────────────┤
│                                                                               │
│  分摊公式：                                                                   │
│  产品制造费用 = 总制造费用 × (产品机器工时 / 总机器工时)                     │

│  2. 按产品分配机器工时                                                       │

│                                                                               │
└──────────────────────────────────────────────────────────────────────────────┘
```

**机器时法分摊SQL：**

```sql
WITH machine_hour_summary AS (
    SELECT 
        mh.production_id,
        mh.style_id,
        SUM(mh.machine_hours) AS total_machine_hours
    FROM jxc_machine_hour mh
    WHERE mh.is_deleted = 0
      AND mh.account_period = :accountPeriod
    GROUP BY mh.production_id, mh.style_id
),
total_machine_hours AS (
    SELECT SUM(total_machine_hours) AS grand_total
    FROM machine_hour_summary
)
SELECT 
    mhs.production_id,
    mhs.style_id,
    s.style_code,
    s.style_name,
    mhs.total_machine_hours,
    (SELECT grand_total FROM total_machine_hours) AS grand_total_hours,
    :totalManufacturingCost * 
        (mhs.total_machine_hours / (SELECT grand_total FROM total_machine_hours)) 
        AS allocated_cost
FROM machine_hour_summary mhs
JOIN jxc_style s ON mhs.style_id = s.id
ORDER BY s.style_code;
```

### 35.5 在制品成本核算逻辑

#### 35.5.1 约当产量法

```
┌──────────────────────────────────────────────────────────────────────────────┐
│                            约当产量法                                         │
├──────────────────────────────────────────────────────────────────────────────┤
│                                                                               │
│  基本原理：                                                                   │
│  将在制品按完工程度折算为相当于完工产品的数量                                │
│                                                                               │
│  约当产量 = 在制品数量 × 完工程度                                            │
│                                                                               │
│  完工程度确定：                                                               │
│  - 材料：按投料程度（一次投料/逐步投料）                                     │
│  - 人工：按加工程度                                                          │
│  - 制造费用：按加工程度                                                      │
│                                                                               │
│  成本分配公式：                                                               │
│  单位成本 = (期初在制品成本 + 本期发生成本) / (完工产品数量 + 约当产量)      │
│  完工产品成本 = 完工产品数量 × 单位成本                                      │
│  期末在制品成本 = 约当产量 × 单位成本                                        │
│                                                                               │
└──────────────────────────────────────────────────────────────────────────────┘
```

**成本差异计算SQL：**

```sql
WITH actual_cost AS (
    SELECT 
        production_id,
        style_id,
        SUM(material_cost) AS actual_material,
        SUM(labor_cost) AS actual_labor,
        SUM(overhead_cost) AS actual_overhead
    FROM jxc_cost_collect
    WHERE is_deleted = 0
      AND account_period = :accountPeriod
    GROUP BY production_id, style_id
),
standard_cost AS (
    SELECT 
        po.id AS production_id,
        po.style_id,
        po.output_quantity * sb.material_cost AS std_material,
        po.output_quantity * sb.labor_cost AS std_labor,
        po.output_quantity * sb.overhead_cost AS std_overhead
    FROM jxc_production_order po
    JOIN jxc_standard_cost sb ON po.style_id = sb.style_id
    WHERE po.is_deleted = 0
      AND po.account_period = :accountPeriod
)
SELECT 
    ac.production_id,
    s.style_code,
    s.style_name,
    ac.actual_material,
    sc.std_material,
    ac.actual_material - sc.std_material AS material_variance,
    ac.actual_labor,
    sc.std_labor,
    ac.actual_labor - sc.std_labor AS labor_variance,
    ac.actual_overhead,
    sc.std_overhead,
    ac.actual_overhead - sc.std_overhead AS overhead_variance,
    (ac.actual_material + ac.actual_labor + ac.actual_overhead) - 
    (sc.std_material + sc.std_labor + sc.std_overhead) AS total_variance
FROM actual_cost ac
JOIN standard_cost sc ON ac.production_id = sc.production_id
JOIN jxc_style s ON ac.style_id = s.id
ORDER BY ABS(total_variance) DESC;
```
