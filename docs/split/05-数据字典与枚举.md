## 十七、数据字典与枚举

### 17.1 枚举命名规范

```
命名规范:
- 枚举常量: 大写下划线格式 (PENDING, IN_PROGRESS, COMPLETED)
- 数据库值: 小写下划线格式 (pending, in_progress, completed)
- 显示名称: 中文名称，通过前端映射

枚举定义格式:
| 枚举值 | 数据库值 | 显示名称 | 说明 |
```
