## 一、高优先级任务

### 1.1 创建工序异常数据库表

**问题描述**: `ProcessException` 实体类存在，但数据库表未创建

**修复步骤**:
1. 创建迁移脚本 `V2026.04.02__process_exception.sql`
2. 定义表结构 `jxc_process_exception`
3. 执行迁移脚本

**文件清单**:
- 新建: `backend/src/main/resources/db/migration/V2026.04.02__process_exception.sql`

**表结构设计**:
```sql
CREATE TABLE jxc_process_exception (
    exception_id BIGINT PRIMARY KEY AUTO_INCREMENT,
    bundle_id BIGINT,
    bundle_no VARCHAR(50),
    process_id BIGINT,
    process_name VARCHAR(100),
    worker_id BIGINT,
    worker_name VARCHAR(50),
    exception_type VARCHAR(50),
    description TEXT,
    status VARCHAR(20) DEFAULT 'PENDING',
    handle_result TEXT,
    handler_id BIGINT,
    handler_name VARCHAR(50),
    handle_time DATETIME,
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP,
    update_time DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    deleted TINYINT DEFAULT 0
);
```
