### 5.2 修复建议

#### P0 - `jxc_product_spu` 实体类字段过多 (N1)

**问题**: 实体类 `ProductSpu.java` 有10个字段在数据库中不存在：`image_urls`, `purchase_price`, `retail_price`, `wholesale_price`, `exchange_points`, `default_warehouse_id`, `product_location`, `warning_qty`, `attr1_name`, `attr2_name`

**方案** (二选一):
1. **修改数据库** - 在 `jxc_product_spu` 表添加缺失列
2. **修改实体类** - 删除或标记 `@TableField(exist = false)` 这些多余字段

**推荐方案1**，因为商品价格和图片是核心业务字段。

```sql
ALTER TABLE jxc_product_spu
  ADD COLUMN image_urls TEXT COMMENT '商品图像' AFTER spec,
  ADD COLUMN purchase_price DECIMAL(10,2) DEFAULT NULL COMMENT '进货价格' AFTER brand_id,
  ADD COLUMN retail_price DECIMAL(10,2) DEFAULT NULL COMMENT '零售价格' AFTER purchase_price,
  ADD COLUMN wholesale_price DECIMAL(10,2) DEFAULT NULL COMMENT '批发价格' AFTER retail_price,
  ADD COLUMN exchange_points INT DEFAULT NULL COMMENT '兑换积分' AFTER wholesale_price,
  ADD COLUMN default_warehouse_id BIGINT DEFAULT NULL COMMENT '默认仓库ID' AFTER exchange_points,
  ADD COLUMN product_location VARCHAR(100) DEFAULT NULL COMMENT '商品货位' AFTER default_warehouse_id,
  ADD COLUMN warning_qty INT DEFAULT NULL COMMENT '库存预警' AFTER product_location,
  ADD COLUMN attr1_name VARCHAR(50) DEFAULT NULL COMMENT '属性1名称' AFTER remark,
  ADD COLUMN attr2_name VARCHAR(50) DEFAULT NULL COMMENT '属性2名称' AFTER attr1_name;
```

#### P1 - 其他实体类字段不匹配 (N2/N3/N4)

**N2 - jxc_style**: 
```sql
ALTER TABLE jxc_style
  ADD COLUMN designer_id BIGINT DEFAULT NULL COMMENT '设计师ID' AFTER brand_id,
  ADD COLUMN target_age_group VARCHAR(20) DEFAULT NULL COMMENT '目标年龄段' AFTER target_gender,
  ADD COLUMN tech_pack VARCHAR(500) DEFAULT NULL COMMENT '工艺包' AFTER sample_image;
```

**N3 - jxc_size_category**: 修改实体类 `SizeCategory.java` 的 `name` 字段为 `categoryName` + `@TableField("category_name")`

**N4 - jxc_aql_standard**: 修改实体类或数据库列名使其匹配

**N5 - jxc_factory**: 创建表或从其他地方获取建表SQL

#### P1/P2 - 权限标识不匹配 (N6-N9)

**统一修复方案**: 将Controller中所有权限标识统一加 `data:` 前缀，与数据库保持一致：
- `bom:view` → `data:bom:view`
- `production:order:view` → `data:production:order:view`
- `mes:quality:view` → `data:mes:quality:view`
- 其他所有权限标识

或修改数据库 `jxc_sys_menu.perms` 列去除 `data:` 前缀。
