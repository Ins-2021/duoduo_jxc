### 2.2 创建销售订单 - 错误分析

**请求**:
```json
POST /sales
{
  "orderType": 1,
  "customerId": 1,
  "orderDate": "2026-03-31",
  "details": [{
    "skuId": 1,
    "qty": 10,
    "unitPrice": 100.00,
    "warehouseId": 1
  }]
}
```

**错误信息**:
```
Field 'sku_id' doesn't have a default value
Field 'spu_id' doesn't have a default value
```

**根本原因**: `jxc_sales_order_detail` 表结构与实体类不匹配，缺少 `sku_id` 和 `spu_id` 字段定义。

### 2.3 审核销售订单 - 错误分析

**错误信息**:
```
Unknown column 'level' in 'field list'
```

**根本原因**: `jxc_customer` 表缺少 `level` 字段。
