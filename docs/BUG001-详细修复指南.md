# B001 Bug详细修复指南

## 问题描述
PurchaseOrderServiceImpl 第238行硬编码 status=1，应使用枚举

## 当前代码
```java
// backend/src/main/java/com/duoduo/jxc/service/impl/PurchaseOrderServiceImpl.java:238
stockInOut.setStatus(1); // 直接已审核
```

## 修复步骤

### 1. 确认StockInOutStatusEnum已存在
文件：`backend/src/main/java/com/duoduo/jxc/enums/StockInOutStatusEnum.java`

确保枚举包含：
- DRAFT(0, "草稿")
- PENDING(1, "待审核") 或 APPROVED(1, "已审核")
- 根据实际业务确认状态码含义

### 2. 修改PurchaseOrderServiceImpl

找到第238行，替换为：
```java
// 修改前
stockInOut.setStatus(1); // 直接已审核

// 修改后
stockInOut.setStatus(StockInOutStatusEnum.APPROVED.getCode());
```

### 3. 添加import语句
```java
import com.duoduo.jxc.enums.StockInOutStatusEnum;
```

### 4. 验证编译
```bash
cd /Users/zxh/mycoding/duoduo_jxc/backend && mvn compile -q
```
