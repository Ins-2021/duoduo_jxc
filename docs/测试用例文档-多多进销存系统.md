# 多多进销存系统 - 全面测试用例文档

**测试日期**: 2026-03-30
**测试环境**: darwin (macOS)
**系统版本**: 主版本

---

## 一、测试范围概述

本系统是一套进销存+ERP+计件工资管理系统，主要包含以下功能模块：
- 基础资料管理（商品、客户、供应商、仓库、账户）
- 销售管理（批发、零售、预订单、销售退货）
- 采购管理（进货、预购、退货）
- 库存管理（查询、预警、盘点、出入库、调拨）
- 财务管理（应收、应付、核销、收支）
- 计件工资管理（计件记录、工资单）
- 生产管理（裁床、工艺、质检、生产工单）
- 报表统计（利润分析、生产报表、成本报表）
- 系统管理（用户、角色、权限、日志）
- 小程序端（老板、裁床、质检、组长、经理角色）

## 六、测试总结要点

测试完成后需要总结以下内容：
1. **测试执行概况**：测试用例总数、执行数、通过数、失败数
2. **问题明细**：发现的所有问题列表，包含问题描述、严重程度、优先级
3. **功能覆盖情况**：各模块测试覆盖率
4. **遗留问题**：未解决问题列表
5. **测试建议**：改进建议
