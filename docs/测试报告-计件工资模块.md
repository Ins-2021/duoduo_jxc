| 项目 | 内容 |
|-----|------|

| **请求体** | `{"yearMonth":"2026-03","startDate":"2026-03-01","endDate":"2026-03-31","departmentId":1,"departmentName":"生产部","employeeCount":1,"totalBaseWage":3000.00,"totalPieceWage":825.00,...}` |
| **预期结果** | 返回新建的工资单ID |
| **实际结果** | `{"code":500,"msg":"系统异常，请稍后重试: SQL语法错误}`` |
| **测试结果** | **失败** |

### 4.3 工资单问题分析

**问题描述**: 新增工资单时出现SQL语法错误

**错误信息**:
```
You have an error in your SQL syntax; check the manual that corresponds to your MySQL server version for the right syntax to use near 'year_month, start_date, end_date, department_id, department_name, employee_count' at line 1
```

**问题原因**:
1. `year_month` 是MySQL的保留字/关键字
2. MyBatis在生成INSERT语句时未对保留字进行转义
3. 实体类字段与数据库表字段命名映射存在问题

**解决方案建议**:
1. 修改实体类，使用 `@TableField("year_month")` 注解显式指定字段映射
2. 或者重命名数据库字段，避免使用保留字
3. 或者在MySQL配置中启用 `ANSI_QUOTES` 模式
