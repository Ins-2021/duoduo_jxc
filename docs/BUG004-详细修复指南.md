# B004 Bug详细修复指南

## 问题描述
单号前缀、状态码等硬编码问题

## 涉及文件
1. `PurchaseOrderServiceImpl.java` - 单号前缀 "RK"
2. `QuotationServiceImpl.java` - 单号前缀 "BJ", 状态 "draft/accepted"

## 修复步骤

### 1. PurchaseOrderServiceImpl.java

**当前代码（约234行）**：
```java
stockInOut.setBillNo("RK" + LocalDate.now().format(DateTimeFormatter.ofPattern("yyyyMMdd"))
        + UUID.randomUUID().toString().replace("-", "").substring(0, 6).toUpperCase());
```

**修复方案**：
```java
// 方案A：使用常量
private static final String STOCK_IN_PREFIX = "RK";
private static final String STOCK_OUT_PREFIX = "CK";

// 方案B：接入单号规则模块（推荐）
@Autowired
private BillNoRuleService billNoRuleService;

String billNo = billNoRuleService.generateBillNo("stock_in"); // 入库单
// 或
String billNo = billNoRuleService.generateBillNo("stock_out"); // 出库单
```

### 2. QuotationServiceImpl.java

**当前代码**：
```java
// 单号前缀硬编码
String quotationNo = "BJ" + ...;

// 状态硬编码
quotation.setStatus("draft");
// 或
quotation.setStatus("accepted");
```

**修复方案**：
```java
// 1. 单号使用规则模块或常量
private static final String QUOTATION_PREFIX = "BJ";

// 2. 状态使用枚举
quotation.setStatus(QuotationStatusEnum.DRAFT.getCode());
quotation.setStatus(QuotationStatusEnum.ACCEPTED.getCode());
```

## 需要创建的枚举（如果不存在）

### QuotationStatusEnum.java
```java
package com.duoduo.jxc.enums;

import lombok.Getter;

@Getter
public enum QuotationStatusEnum {
    DRAFT(0, "草稿", "draft"),
    PENDING(1, "待确认", "pending"),
    ACCEPTED(2, "已接受", "accepted"),
    REJECTED(3, "已拒绝", "rejected"),
    EXPIRED(4, "已过期", "expired");

    private final Integer code;
    private final String desc;
    private final String value; // 用于前端展示或数据库存储

    QuotationStatusEnum(Integer code, String desc, String value) {
        this.code = code;
        this.desc = desc;
        this.value = value;
    }
}
```
